# 💬 Interactive Comments Section

A fully functional interactive comments section built with vanilla HTML, CSS, and JavaScript.

👉 [View Live](https://mansityagi548.github.io/interactive-comments-section/)


## 📸 Preview

![Desktop Preview](preview.jpg)

## ✨ Features

- 🗨️ **Add comments** — Post new comments as the current user
  
- 💬 **Reply to comments** — Reply to any comment or reply in a thread
  
- ✏️ **Edit your comments** — Edit any comment or reply you've made
  
- 🗑️ **Delete your comments** — Delete with a confirmation modal so you don't accidentally remove anything

  
- 👍 **Upvote / Downvote** — Vote on any comment (votes don't go below 0)
  
- 🕒 **Dynamic timestamps** — Shows "just now", "2 days ago", "1 month ago" etc automatically
  
- 💾 **Persistent data** — Comments are saved to localStorage so they survive page refresh
  
- 📱 **Fully responsive** — Works on mobile, tablet, and desktop
  

## 🛠️ Built With

- HTML
- CSS
- Vanilla JavaScript (ES6 Modules)




## 🚀 How to Run Locally

1. Clone the repo
```bash
git clone https://github.com/mansityagi548/interactive-comments-section.git
```

2. Open `index.html` in your browser or use a live server

> No dependencies or build tools needed — pure vanilla JS!
